Our Project

This prototype webapplication has been created for the class 'Skills: Programming with Advanced Computer Languages' at the University of St. Gallen. The entire project can be broken down into three main parts:

  • A Webscrapper to pull information about real estate listings from Immoscout24.ch using the Scrapy framework
  • A Random Forest trained on the scraped listings to predict a property price based on user input using scikit-learn
  • The Webapplication you are seeing right now which has been programmed using the Flask and Dash frameworks
The data utilized in this project has been scraped in April 2024. To be fit for modelling and the implementation into the webApp, the data has been cleaned along the process.
